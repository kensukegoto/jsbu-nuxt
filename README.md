# コミット

## ルール

```
ADD  機能追加など
CHN  関数名変更・ロジック変更など（機能追加を伴わない変更）
FIX  修正
DEL  ファイル削除・関数削除など
DOC  README
INI  初期化
```

## 履歴

```
INI  命名規則策定・Sass導入・デフォルトCSS設定
ADD  ロゴ配置
ADD  GNavi作成
```
